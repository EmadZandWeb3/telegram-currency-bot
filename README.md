# 💱 Telegram Currency Bot

A simple Telegram bot that periodically fetches currency prices and sends updates automatically.

## 🚀 Features

- Fetches live currency prices from API
- Sends automatic updates to Telegram
- Supports scheduled sending (e.g. every 5 minutes)
- Uses environment variables for security

---

## 🛠 Tech Stack

- Node.js
- node-telegram-bot-api
- Axios
- node-cron (for scheduling)
